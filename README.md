

# 🔐 Password Manager App

A simple yet practical password management application built in Python to securely store and retrieve user credentials.

This project demonstrates structured data handling, file persistence, and clean Python application logic.

---

## 📌 Overview

The Password Manager allows users to:

- Save website credentials (website, email/username, password)
- Store data persistently using JSON
- Retrieve saved credentials
- Generate secure random passwords
- Practice structured file handling

This project focuses on building strong foundations in Python application development.

---

## 🛠 Tech Stack

- Python
- JSON (data storage)
- CSV (data experimentation)
- File I/O
- Jupyter Notebook (experiments & testing)

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/asfandyar-prog/password-App.git
```

2. Navigate into the project directory:

```bash
cd password-App
```

3. Run the main Python file:

```bash
python main.py
```
